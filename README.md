# Paradise Nursery 🌱

A React + Redux based shopping cart web application for an online plant store.

## 🚀 Features
- Browse plants categorized into different sections
- View plant name, price, and image
- Add plants to cart
- Increase / decrease quantity
- Remove items from cart
- View total cart value
- Checkout functionality (clears cart)

## 🛠️ Tech Stack
- React (Vite)
- Redux Toolkit
- React Router DOM
- CSS

## 📂 Project Structure
- `src/components` → UI Components
- `src/redux` → State management
- `src/App.jsx` → Main app
- `src/App.css` → Styling
